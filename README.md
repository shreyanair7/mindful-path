# MindAnchor

MindAnchor is a holistic mental health support web application designed to help users understand emotional stress patterns, access immediate coping support, and track mental wellbeing over time.

The platform focuses on long-term awareness and resilience rather than one-time conversations, providing users with a stable and supportive mental wellness companion.

---

## What MindAnchor Does

- Enables safe emotional expression and guided reflection  
- Helps users identify stress patterns through structured inputs  
- Provides immediate coping and grounding exercises  
- Tracks mental wellness trends over time  
- Encourages professional or crisis support when needed  
- Supports continuous engagement rather than isolated interactions  

---

## Core Experience

MindAnchor follows a structured flow centered on awareness, support, and progress.

### Secure Access

- Users sign up and log in securely  
- Sessions persist across refreshes  
- Core features are available only to authenticated users  

### AI Companion

- Interactive companion for guided self-expression  
- Encourages users to articulate thoughts and emotions  
- Designed for reflection and support, not diagnosis  

### Stress and Wellness Input

- Users describe their current mental state  
- Inputs are used to track changes over time  
- Enables personalized insights based on user history  

### Progress and Wellness Trends

- Visualizes past interactions and wellness patterns  
- Helps identify improvement or recurring stress cycles  

### Coping and Grounding Exercises

- Immediate stress-relief exercises  
- Includes breathing techniques and grounding practices  

### Crisis Support Resources

- Dedicated access to mental health and crisis support resources  
- Guides users toward professional help when required  

---

## Application Flow

- Users log in and access a personalized dashboard  
- From the dashboard, users can:
  - Interact with the AI Companion  
  - Provide wellness check-ins  
  - View progress and trends  
  - Engage in coping exercises  
  - Access crisis support resources  

- The experience adapts over time based on user engagement and interaction history  

---

## Technology Stack

### Frontend

- React  
- TypeScript  
- Vite  
- Tailwind CSS  

### Routing and State Management

- React Router  
- Context API  

### Backend and Services

- FastAPI for backend services  
- Python for data handling and model integration  
- Supabase for authentication and secure data storage  

### AI and Machine Learning

- NLP-based emotional stress analysis  
- Lightweight ML models for stress signal extraction  
- Adapter-based fine-tuning for personalized model adaptation (future-ready)  

---

## Privacy and Ethics

- Designed with user safety and privacy as core principles  
- User inputs are handled securely and with consent  
- The platform does not provide medical or clinical diagnoses  
- Intended to support awareness and wellbeing, not replace professional care
